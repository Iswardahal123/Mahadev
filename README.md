# Mahadev Kirtan Sangh 🕉️

## Vercel Pe Deploy Kaise Karein

### Option 1: GitHub se (Recommended)
1. Is folder ko GitHub pe upload karo
2. [vercel.com](https://vercel.com) pe jao
3. "New Project" → GitHub repo select karo
4. Framework: **Create React App** select karo
5. Deploy karo ✅

### Option 2: Vercel CLI se
```bash
npm install -g vercel
cd mahadev-kirtan-sangh
npm install
vercel
```

## Local Chalane Ke Liye
```bash
npm install
npm start
```
